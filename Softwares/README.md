# Softwares

Esta pasta e reservada para ferramentas locais, SDKs e instaladores usados no desenvolvimento do AegisRover.

O conteudo binario desta pasta nao e versionado no Git para manter o repositorio leve e adequado ao GitHub. Para recompor o ambiente local, baixe ou reinstale os pacotes oficiais conforme a necessidade:

- Bosch BSEC / BSEC2 Library;
- Bosch BME68x Library;
- BME AI-Studio Desktop;
- bibliotecas Arduino/ESP32 relacionadas ao BME688.

Arquivos extraidos de instaladores, executaveis, DLLs, bibliotecas compiladas e pacotes compactados devem permanecer apenas no ambiente local.
