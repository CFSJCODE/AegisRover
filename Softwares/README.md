# Softwares

Esta pasta é reservada para ferramentas locais, SDKs e instaladores usados no desenvolvimento do AegisRover.

O conteúdo binário desta pasta não é versionado no Git para manter o repositório leve e adequado ao GitHub. Para recompor o ambiente local, baixe ou reinstale os pacotes oficiais conforme a necessidade:

- Bosch BSEC / BSEC2 Library;
- Bosch BME68x Library;
- BME AI-Studio Desktop;
- bibliotecas Arduino/ESP32 relacionadas ao BME688.

Arquivos extraídos de instaladores, executáveis, DLLs, bibliotecas compiladas e pacotes compactados devem permanecer apenas no ambiente local.
